Project 4: Titanic Shiny App Prompts

Prompt 1 - Initial Shiny App Prompt
-----------------------------------
Build a single-file R Shiny app named app.R for my Project4_Titanic_Shiny folder.

Context:
- I already have Tidymodels code from my midterm assignment for Titanic survival prediction.
- The project goal is to build a Shiny front end that uses Tidymodels code to train and compare models predicting Titanic passenger survival.
- Candidate models:
  1. Null model
  2. kNN
  3. Boosted C5.0
  4. Random Forest
  5. Regularized Logistic Regression
  6. Naive Bayes
- Response variable: Survived
- Use the titanic training data and compare models using validation performance.
- Use accuracy as the main model selection metric.

Requirements:
1. Create a runnable single-file app.R.
2. Use shiny, tidyverse, and tidymodels.
3. Organize the app into tabs:
   - Overview
   - Data Preparation
   - Train Models
   - Model Comparison
   - Best Model
   - Predict New Passenger
4. Add a button so models train only when clicked.
5. Show:
   - preprocessing summary
   - table of model metrics
   - best model selection
   - confusion matrix for the best model
   - ROC plot if possible
6. Add a small user input form for a new passenger and output the predicted survival class from the best model.
7. Handle missing values safely.
8. Make sure Survived is treated as a factor with levels No and Yes.
9. Keep code readable and well-commented.
10. Do not depend on external objects already loaded in memory.
Return complete runnable code only.
